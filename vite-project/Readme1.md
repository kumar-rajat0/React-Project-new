# ShoppyGlobe Backend API

## Technologies Used
- Node.js
- Express.js
- MongoDB
- JWT Authentication
- ThunderClient


## How to Run the Project
1. Clone the repository
2. Install dependencies:
   npm install
3. Start server:
   npm run dev

   ## API Endpoints
- GET /api/products
- GET /api/products/:id
- POST /api/auth/register
- POST /api/auth/login
- POST /api/cart
- PUT /api/cart/:id
- DELETE /api/cart/:id
