**Project overview** : ShoppyGlobe is a basic e‑commerce application where users can:
                        1). Browse products
                        2), View product details
                        3). Search products
                        4). Add/remove items from the cart
                        5). Adjust cart quantities
                        6). Proceed to checkout
                        7), Place a dummy order
